CurveDNS

A DNSCurve Forwarding Name Server

This Github pages website is only here to deliver the Master Thesis Shaping DNS Security with Curves by Harm van Tilborg. This thesis was written along with the creation of the CurveDNS software and gives insight in a comparative security analysis of both DNSSEC and DNSCurve. Besides, it also discusses some of the design decisions of CurveDNS and its performance.
